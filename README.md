#  Password Strength Analyzer

A Python-based tool that evaluates the strength of user-entered passwords and suggests stronger alternatives.

---

##  Features
- Checks password length and complexity  
- Validates use of uppercase, lowercase, numbers, and special characters  
- Suggests strong and secure passwords  
- Prevents reuse of previously used passwords (basic implementation)  

---

##  Technologies Used
- Python  
- Regular Expressions (re module)  
- Random & String modules
